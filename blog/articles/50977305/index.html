このように、 Google Code Project Hosting では HTML や画像ファイルなどもアップロードでき、ブラウザで直接表示させることができます。プロジェクトのホームページやドキュメント公開に便利ですね。この方法には以下のようなメリットがあります。

  • 履歴管理をはじめとした Subversion の各種機能が利用できる。
  • プロジェクトのソースファイルなどと同じリポジトリで統一的に管理できる。
  • 複数メンバーによる共同作業が可能。メンバーが世界中に散らばっていても問題なし！
  • 全文検索、 Issue Tracking などの Google Code Project Hosting の各種機能が利用できる。
  • 画像など、テキスト以外のファイルの公開も可能。
  • Google Personalized Homepage ガジェット、 Firefox 検索プラグインなどを直接デプロイできる。

ただし、アップロードしたファイルは選択したオープンソースライセンスが強制的に適用されると考えられますので、ライセンスに適合しないファイルをアップロードしないように、くれぐれも注意してください。
