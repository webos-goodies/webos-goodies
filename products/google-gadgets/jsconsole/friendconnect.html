以下のコードだけでviewerとownerのデータがログに表示されます。ログはコンソールの「Log」タブで見れます。 
osapi.newBatch().
  add('viewer', osapi.people.getViewer()).
  add('owner', osapi.people.getOwner()).
  execute(function(result) {
    jsconsole.log('Viewer name : ' +
      result.viewer.displayName + '\n');
    jsconsole.log('Owner name  : ' +
      result.owner.displayName) + '\n';
  });
