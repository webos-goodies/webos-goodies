WebOS にとって、 Web ブラウザはその可能性を決定付ける極めて重要なアプリケーションです。 Web 標準を正しくサポートすることはもちろん、堅牢なセキュリティー、高いパフォーマンス、柔軟な拡張性、幅広いプラットフォームのサポートなど、 WebOS 時代の Web ブラウザに対する用件は非常に高度なものとなるでしょう。そのような用件を満たせるブラウザとして、 WebOS Goodies は Opera と Firefox の 2 つを推奨しています。その一環として、このカテゴリでは Opera と Firefox を活用するためのさまざまな情報をご紹介し、微力ながらこれらのブラウザの普及を後押ししていきたいと考えています。
