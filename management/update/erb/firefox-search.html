Firefox 検索プラグインの作り方

Firefox のアドレスバーの横にある検索ボックス、実は自由に拡張できることをご存知でしょうか。Firefox検索プラグインを利用することで、好きな検索サイトを追加することができます。このコーナーでは、そのFirefox検索プラグインを自分で作成する方法をご紹介しています。簡単な定義ファイルを書くだけなので、いったん作り方を覚えてしまえば、必要な検索プラグインを Web で探すよりも楽なくらいですよ。自分の Web サイト用の検索エンジンを Google Customize Search で作って、そのプラグインを配布するのも良いアイディアですね。 Firefox 2.0 からはサジェスト機能が追加され、より面白い使い方が考えられるようになっています。 Firefox をご使用の皆さん、基礎知識として押さえておくのはいかがでしょうか。
