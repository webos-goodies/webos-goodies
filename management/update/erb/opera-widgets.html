Opera Widgets の作り方

Opera 9 で追加されたさまざまな機能の中でも異彩を放っているのが Opera Widgets です。 HTML や JavaScript といった既存技術を利用してデスクトップで動作するミニアプリケーションを作成できる機能で、その仕様は W3C で策定中の Widgets 1.0 規格のベースにもなっています。このコーナーでは、その Opera Widgets の作成方法をご紹介しています。自分で好みのウィジェットが作れるほか、 XMLHttpRequest が任意のサイトに対して発行できるため、各種 Web API を利用したマッシュアップの実験も手軽に行えますね。 Opera をご利用の皆さん、この素晴らしい機能をぜひともご活用ください。
