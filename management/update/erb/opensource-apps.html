<%= escapeHTML(@captions['net2ftp']) %>

SFTP などの暗号化プロトコルが開発された現在でも、 FTP はまだまだ健在。とくに安価なレンタルサーバーのほとんどは FTP を主なファイル転送手段として提供しています。しかし、洗練された Web アプリケーションと比べて FTP クライアントの使いにくいこと！真の WebOS フリークならば、 FTP といえども Web 上で活用したいところです。そんな貴方にお勧めなのが、この net2ftp 。 これは PHP で実装された FTP クライアントで、オープンソースソフトウェアとして公開されています。 Web ブラウザから自由に FTP が使えるのはもちろん、 WYSIWYG な HTML エディタや色分け表示機能付きのコードエディタも内蔵されており、まるでちょっとした CMS のような感覚で FTP サーバー上のファイルを編集・管理できます。さあ、貴方も net2ftp で快適な Web & FTP ライフを送りましょう！
